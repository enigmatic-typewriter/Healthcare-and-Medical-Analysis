Internship Project

Diabetes risk analysis, cohort exploration, and early prediction in one web app.

This project uses the Kaggle diabetes prediction dataset to study disease prevalence, compare patient risk factors, and estimate diabetes likelihood from clinical indicators such as HbA1c, BMI, hypertension, and blood glucose level.

Method

Pipeline

  1. Load CSV data from Kaggle export or demo sample.
  2. Clean numeric values and normalize category labels.
  3. Engineer feature vectors from demographic and clinical fields.
  4. Train a logistic regression model in the browser.
  5. Evaluate accuracy, precision, recall, and F1-score.
  6. Use the fitted weights to estimate a new patient risk score.

Internship Value

Why this project is useful

Shows EDA on healthcare data with meaningful business and clinical insights.

Connects risk indicators like BMI, HbA1c, glucose, and comorbidities to a prediction problem.

Runs with zero backend so the project is easy to demo, deploy, and review on GitHub.

Can be extended later with Python notebooks, Flask APIs, or deep learning models.
